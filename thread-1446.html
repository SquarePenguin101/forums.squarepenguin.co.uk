04-07-2017, 10:16 AM
The hidden folder (presumably C:\Users\ElizabethsPC\.get_iplayer) is get_iplayer's profile directory, where settings and cache files are stored. Since removing it solved your problem, that confirms that the permissions on that folder were incorrect. You should find that the profile directory has been re-created in the same location. It doesn't live in AppData unless you explicitly configure get_iplayer to put it there.
