How do I download 28/11/2017 Newsnight without BBC News banner?

user-1773

29-11-2017, 01:54 PM
Hi I'm trying to download a programme (newsnight) broadcast on BBC Two, but when I download it, it is shown as being from the "BBC News" Channel and has an annoying large red BBC News banner at the bottom of the video. However when I watch it online on the iplayer website it is shown without any banner.

Is there some way I can tell get_iplayer to download the BBC Two version which plays on the website and not the BBC News version of the programme with the banner? I've tried giving the url as an argument:
https://www.bbc.co.uk/iplayer/episode/b0...t-28112017
but it still downloads the same BBC News bannered version.
Any ideas?
Many thanks!

user-2

29-11-2017, 03:24 PM
Use a different value for --tvmode. For some reason, the default "hls" streams show the banner for that episode.
