Radio Program not showing

user-820

10-01-2015, 12:08 PM
Hello, I recently installed get-iplayer in order to download some DRM free content for offline listening at my leisure. Yesterday evening a program, ID: "b04wmklh", came on that I now want to download. When I searched in get-iplayer I only get previous episodes of this radio series which were apparently uploaded at the exact time when I installed the program. This makes me think I need to refresh the list, but I can't figure out how to do this. Any advice? Thanks!

user-585

10-01-2015, 06:33 PM
You need to run
Code:
--refresh
with your search command to get programmes that have appeared since the last refresh was done. By default it will do this automatically every 4 hours when you run a command. The refresh process will take a few minutes.
