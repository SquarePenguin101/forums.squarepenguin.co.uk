If you look up the info on the PID using get_iplayer --pid b075f6my --info then you will be able to see there are several versions.

Assuming that the most recently changed version is the 'correct' version that replaces the 34 minute programme, you can see that 'editorial' is the version you need to download.

To select a version of a programme to download, you use the --versions flag.

A complete command would be:

get_iplayer --pid b075f6my --versions=editorial --force --overwrite
