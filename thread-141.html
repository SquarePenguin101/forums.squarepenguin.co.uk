03-04-2014, 11:50 PM
Only the signed version of Episode 3 is available to get_iplayer, even though both versions are available on the iPlayer site. This is one of those corner cases where the default version disappears from the data feeds used by get_iplayer (because the usual 7-day expiry is triggered), but the signed version has a longer life, so they remain on the iPlayer site as a linked pair.

You can indeed search for the PID found in the URL. Just remember to set the "Search in" field to "Pid".

Alternatively, use the "Programme Version" field to bring signed programmes into your search results. For the format, see the description of the
Code:
--versions
option in:

https://squarepenguin.co.uk/wiki/manpage/
