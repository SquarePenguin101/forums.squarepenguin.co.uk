09-05-2018, 05:28 PM
Run chcp 1252 before running get_iplayer to change console code page for current session. Then use --encoding-locale=cp1252 --encoding-locale-fs=cp1252 when running get_iplayer. get_iplayer is not intended for use with non-Western system locales, so no guarantee that this will work.
