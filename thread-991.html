Have a read of this topic /thread-982.html.

Red button and web only content from the Olympics is available in a limited number of recording modes.

As you can see from the log that you posted, there are no flash streams available for that event. The error message suggests using hls modes instead. However, there are some pretty major problems with those at the moment.

To download this you will need to use one of the hvf download modes but you need to update to version 2.95 of get_iplayer first. hvf modes are not supported in 2.94.
