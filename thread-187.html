02-08-2014, 10:23 PM
Adding get_iplayer installation dir to PATH won't help. Needless to say, rtmpdump runs just fine on 64-bit Windows. I have no idea what was messed up on your particular system that would cause rtmpdump to be misidentified as a 16-bit application. Maybe you have a misbehaving anti-virus, though I don't know how that would cause this particular error. get_iplayer obviously doesn't touch any system components, so upgrading or re-installing should be irrelevant. Only you will know what you've done to your system. One person reported this same problem to the mailing list, then claimed to have found a solution. However, when asked she refused to supply that solution, so no further light was shed on the problem. The only other suggestion I can offer is to try some different rtmpdump binaries. One alternative (there are no doubt others floating around):

http://ffmpeg.zeranoe.com/forum/viewtopi...=29&t=1024
