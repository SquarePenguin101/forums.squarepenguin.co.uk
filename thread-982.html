Olympics: Use hvf or flash recording modes

user-2

05-08-2016, 08:39 PM
As has been discussed elsewhere in the forums, the BBC is currently producing corrupt media for many programmes for the default hls recording modes, with no indication that the problem will be fixed. That includes the Olympics opening ceremony and other Olympics broadcasts. For Olympics programmes, use hvf recording modes instead.

Use --tvmode=hvf if you're able to take the 2GB/hr hit of 1280x720 50fps 5Mbps HD downloads. For the next step down, use --tvmode=hvfsd,hvfbetter to limit the maximum quality to 960x540 50fps 3Mbps. The next step down after that would be --tvmode=hvfbetter to limit the maximum quality to 960x540 25fps 1.7Mbps. If you plan on recording many Olympics programmes, you will want to set --tvmode in your preferences.

It also appears that not every Olympics programme will be available with hvf modes. In that case, use flash modes. Use --tvmode=flash to get up to 1280x720 25fps 2.5Mbps HD (not always available). Use --tvmode=flashbetter to limit the maximum quality to 832x468 25fps 1.5Mbps.

It appears the Olympics programmes shown on main channels (BBC One, Two, Four) are mostly available with all the normal recording modes, so follow the advice above for those programmes. Red button-only programmes mostly appear only to be available with hvf modes.
