There is no additional step. Only you would know how your system got into that state, but it's not worth the effort to diagnose. Uninstall get_iplayer from Control Panel, manually delete the installation directory if not removed during uninstall, manually delete any and all old get_iplayer installations (you appear to have at least one), delete any leftover Start Menu shortcuts and folders related to get_iplayer, then re-install as normal.
