It looks like this programme was recorded live for iPlayer.

When the BBC do this, the programme will usually become available within 15 minutes or so of the actual broadcast finishing. The first version to appear on iPlayer will sort of be a rough edit of the programme which will probably have some of the BBC's introduction on and some adverts afterwards as broadcast. This seems to get processed into all versions of SD streams (both HLS and RTMP) and an HD stream for HLS but not RTMP.

Usually within an hour or so it will be properly trimmed to get rid of most of this overrun from both sides. This will then be processed into all HLS and RTMP stream types including flashhd.

So with anything live or recorded "off air" for iPlayer, you will probably find a bit of a delay before flashhd streams become available.

Most pre-recorded programmes are not done this way so all versions usually become available quicker.
