Firstly you are using an old version of get_iplayer. We are currently up to 2.99. You should update to the newest version.

Secondly the error you are getting is telling you what the problem is:

get_iplayer Output:

WARNING: Required rtmpdump does not exist - cannot download Flash audio/video

You need rtmpdump to download any of the flash modes. It looks like you haven't installed it for some reason.
Using the newest version of get_iplayer you will also be able to use HLS and HVF streams which do not rely on rtmpdump if you don't want to install it.
