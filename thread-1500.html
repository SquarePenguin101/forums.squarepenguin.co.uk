Web PVR: Transferred downloads to new machine, now greyed out in Recordings tab

user-1034

09-09-2017, 11:12 AM
I have installed get_iplayer on a new machine and successfully copied over my PVR searches and download history.

I have also copied over the recordings from the old machine to the folder on the new.  Is there anyway I can refresh the index so the list of recordings in PVR manager are not all greyed out?  Any new recordings I make appear correctly but I cannot see a way to get the old ones to show.

user-2

09-09-2017, 02:04 PM
No. You will need to edit the file paths in the download history to match their new locations, or else place the transferred files in the exact same full path locations as on the old machine.
