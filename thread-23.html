That's the correct site. The installer should attempt to download several files from that site, which you can check individually:

http://www.infradead.org/get_iplayer/latest/get_iplayer
http://www.infradead.org/get_iplayer/lat...player.cgi
http://www.infradead.org/get_iplayer/MANIFEST.v2.83
http://www.infradead.org/get_iplayer/v2....les.plugin
http://www.infradead.org/get_iplayer/v2....ast.plugin

If the problem persists, create an empty directory with the following path:

Code:
C:\Users\<your_username>\.get_iplayer\plugins

I think that should prevent the plugin installation phase from executing and allow the installer to continue. You can download the two .plugin files and place them in that directory later if you wish.
