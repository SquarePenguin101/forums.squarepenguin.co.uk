If you look on the Iplayer web site
  http://www.bbc.co.uk/programmes/b007bxmt
You'll see:
  "This programme is not currently available on BBC Iplayer"

Presumably the BBC haven't acquired the rights to allow this movie to be streamed on Iplayer.
