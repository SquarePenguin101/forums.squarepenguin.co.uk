If you check the 2.95 release notes regarding the 30 day cache it should explain things:

https://github.com/get-iplayer/get_iplay...newchanged

In case it's not clear, the longer you've been running get_iplayer, the 'fuller' it's cache becomes, so a new install only looks back 7 days where as a mature install can look back up to 30.
