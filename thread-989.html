I'd like to use the "raw" option for recording TV (TS) but would like the non-raw (default) for radio recordings. Is this possible? I only see the "raw 1" setting available to go in the "option" file and that does "raw" radio recordings.

user-2

08-08-2016, 12:50 PM
Don't make --raw a global option in that case. Just specify --raw when you create a pvr job for a TV programme and it will only be operative for that pvr job. If you have several TV-only options you always want to use with the pvr, you can put them in a preset and specify the preset when creating a pvr job for a TV programme.
