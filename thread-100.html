rtmpdump will fail sometimes. rtmpdump will crash sometimes. There is nothing you can do about it, and there is nothing get_iplayer can do about it. get_iplayer makes a number of attempts to resume and complete problem downloads, but often it simply won't work. Any number of things can cause rtmpdump problems, but they rarely can be diagnosed and virtually never can be diagnosed remotely. Just delete any partial download and start over.

If rtmpdump crashes inordinately often, you can try an alternate build. See the instructions I gave here:
