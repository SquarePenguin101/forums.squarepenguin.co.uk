Forum Rules

user-2

06-02-2016, 04:28 PM
Below is a list of get_iplayer forum rules. This list is not exhaustive and does not include the basics of civilised nettiquette, which should be considered as required at all times.
  1. Do not request help if you are outside the UK.
  2. Do not post about using VPS servers, VPNs, Smart DNS services, web proxies or any other methods to circumvent geo-blocking measures employed by the BBC.
  3. Do not ask for other users to provide you with copies of any programmes, and do not offer any programmes for download.
  4. Create high quality bug reports. See this thread for a guide.
  5. Make pasted code snippets short and WRAP THEM IN CODE TAGS. To do so, select your text and then hit the code button above the input box. Alternatively you can place a backtick at the start and end of the text you wish to display as code.
  6. Do not post large blobs of code/terminal output into a post. Save such text into a .txt document and upload it as an attachment to your post. The attach button is available under the post input box in the Preview Post screen.
Thank you.
