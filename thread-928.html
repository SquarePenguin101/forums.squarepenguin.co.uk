I am currently struggling with the same issue. I don't know how to get the live, but now I am trying to get the recorded programmes...

You can try this link (for the recorded videos of Glastonbury 
-> http://www.bbc.co.uk/iplayer/episodes/b007r6vx )
and then right click over the video you want to download. Select "Copy link address" and then start to look for the vPID... 

So, the "2016: Blossoms" link is "http://www.bbc.co.uk/iplayer/episode/p03y1gqq/glastonbury-2016-blossoms".

Open this link directly in your browser and then right click the page select "View page source". Then use your browser to find "vpid" in the source text from the BBC iPlayer (I use ctrl+F to do a search in my browser). You will see text that looks something like this (for "2016: James"):


Code:
... "hasDownloads":true,"vpid":"p03y1k36","sq ...


Here the vPID is: "p03y1k36". Once you have the vPID you can download the video with get_iplayer.

Normally, the "masterbrand" for Glastonbury is "BBC_Four", this time it seems to be that the "masterbrand" is "BBC". So, get_iplayer (at least my version) does not seem to find the Glastonbury programmes.

Perhaps it would help to list here the Glastonbury programmes and the vPID to make it nice and easy for everyone?

(24-06-2016, 07:05 PM)How can I get glastonbury live using gip?

e.g. http://www.bbc.co.uk/events/ewcj5v/live/c2rzp6
The live programmes use the red button streams, which are not supported by get_iplayer.  You can feed the HLS master playlists for those streams to VLC or other media players. The URLs look like:

http://a.files.bbci.co.uk/media/live/man...am_XX.m3u8
http://a.files.bbci.co.uk/media/live/man...am_XX.m3u8

where "XX" can be between 01 and 24 (I think) to access different streams. Only a few streams are available at any given time and some will just be showing a "closed" graphic. You'll have to do some trial and error to figure out which stream goes with which Glastonbury stage. At this moment 08-13 appear to be broadcasting.
(24-06-2016, 09:54 PM)So, the "2016: Blossoms" link is "http://www.bbc.co.uk/iplayer/episode/p03y1gqq/glastonbury-2016-blossoms".
With the development version of get_iplayer you can download directly from that URL as normal (was broken in 2.94 by BBC changes), though some warning messages will be printed. Special event programmes sometimes don't have all the metadata that get_iplayer expects.
