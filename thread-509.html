user-2

25-08-2015, 06:20 PM
The temp file is created by AtomicParsley when adding metadata tags to the downloaded file. It may be left on disk if something goes wrong when copying it over the original file or if AP crashes before it is complete. One possible cause is if your output directory is on an a slow network drive. If that is case, use an output directory on a local disk and copy files to final location when download is complete.
