It's possible you have corrupted cache files. Back up and delete the existing cache files (
Code:
%USERPROFILE%\.get_iplayer\*.cache
for Windows,
Code:
$HOME/.get_iplayer/*.cache
for Linux/Unix/OSX), then run get_iplayer again to rebuild them.
