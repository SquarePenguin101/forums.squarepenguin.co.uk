There would be no way to know exactly what you've done to your machine, so it's difficult to even guess what is going on. That error message is generic and doesn't really give a clue what resource is being denied. Even if you're the only user, you don't get away from the system security policies. Try opening the installer via Right Click -> Run As Administrator to ensure that you are invoking with admin privileges. If that doesn't bear fruit, make sure all anti-virus, spyware protection, security software, etc. has been completely stopped, including any associated system services (assuming you're willing to do that) and try again. If that doesn't work, try running the installer from a USB stick or external drive (a bit of long shot).
