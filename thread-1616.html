I think this is something lots of people run into when using the command line. They assume there's something more complex to the process, but it's quite literally just typing in the command all in one go and hitting enter.

That's honestly it.

I'm routinely surprised that more folks don't just simply try it and see what happens, but the failing isn't yours, it's on us with more experience/knowledge in this area to communicate better how simple the command line is. It's quite literally just typing in text and executing it.

As the guide describes, you can build up the command you want with all the things you want to tell get_iplayer to do. Then you just type it in in one go and hit enter. No tricks, that's it.

If you want to save preferences, the guide tells you how, again by simply typing in the relevant command that tells get_iplayer to save a preference, and then hitting enter to execute that 'remember this' command.

Hope that helps...
