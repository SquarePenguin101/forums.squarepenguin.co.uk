How do I restrict cache to only Radio 4 and Radio 4 Extra?

user-1884

22-07-2017, 11:51 AM
Just wondering if you got this sorted.

I've been trying to limit my index to just radio 4 and radio 4 extra using the following command from the documentation:

get_iplayer --prefs-add --refresh-include="Radio 4"

which adds the following line to my options file:

refreshinclude Radio 4

but my index still lists every station.

user-2

22-07-2017, 04:14 PM
Existing cache entries aren't removed when changing setting, but will eventually expire. Use --cache-rebuild (one time only) to wipe the existing cache and repopulate.
