It's not nearly as complicated as it sounds.

All you need to do is go to https://raw.githubusercontent.com/dinkyp...et_iplayer and save that file to your computer.

Then you go and find the get_iplayer folder in Program Files and replace 1 file in there with the new one you have just saved. You might also need to rename that file once you have moved it there.

If you are using Windows, this is the section you need to read https://squarepenguin.co.uk/wiki/install...itheadwin/
