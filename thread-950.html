Hello,

Since updating from 2.94 to 2.96-dev, PVR doesn't seem to work any more. Running PVR via the web interface or by command  line produces an empty response, as if no pvr searches have been added.

The following screenshots show the issues. (1) Adding a PVR search via the web interface seems to add a correct PVR search string.  (2) Now running PVR however creates an empty response, instead of listing the PVR search name i.e. _Midnight_News_name_radio.
