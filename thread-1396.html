Check the release notes about version of Perl over 5.17, I think it was.  Also, there is a missing dependency in the LIBXML perl package for this release.  If you read the dependency list and hunt, you'll learn to "sudo apt-get install libxml-perl"...but you will STILL have an unmet dependency on line seven thousand and something (sorry, don't have the output anymore).

You have to also execute "sudo apt-get install libxml-libxml-perl" for it to work.  Isn't that obvious?  Not a word about it anywhere, so I hope this helps someone.
