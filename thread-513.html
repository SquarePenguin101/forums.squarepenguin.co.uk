There is no need for duplicate postings containing nearly identical output spews.  We only need one.

Those programmes work fine here. One possible explanation for only getting lowest quality is that you're outside the UK and using a proxy.  If that doesn't apply to you, try --modes=hlsbest to avoid rtmpdump.  If there are some Flash servers out there that are incompatible with rtmpdump, there isn't anything we can do about it anyway.
