You say Season 8 is available, not according to what I'm looking at... unless I'm looking in the wrong place... but then you didn't give me a URL, as I requested.

Searching on "Chucklevision" within the BBC iPlayer, I ended up at the following URL:
https://www.bbc.co.uk/cbbc/shows/chucklevision#episodes

According to this there appear to be six episodes currently available from Season 21.

If you right click on each episode in turn, you can save the URL. So, for example, "Cops and Robbers" is at https://www.bbc.co.uk/iplayer/cbbc/episode/b00phvly.

I don't use the pvr but if you open a command prompt and type:
get_iplayer --PID https://www.bbc.co.uk/iplayer/cbbc/episode/b00phvly
or even:
get_iplayer --PID b00phvly
you should see the download taking place. Repeat this process for each programme.

My understanding is that internet only programmes can only be downloaded using the programme PID. I wouldn't have a clue how to do this using the PVR (If indeed you can) as I never use the PVR. I'm a command prompt sorta guy!
