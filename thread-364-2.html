Yes in 2.90 there was sometimes a default and default2 version of programmes and default2 was the only one that worked. This has been fixed in version 2.91 and the current problem which means that you sometimes have to use --playlist-metadata to download a programme should be fixed when 2.92 is released.

You really do need to keep get_iplayer up to date at the moment as there are a lot of changes going on.
