I tried this today, and found 375 Radio Cymru and 310 Radio Wales programmes indexed.  Have you tried a cache rebuild? (yeah, it takes ages...)

get_iplayer --cache-rebuild --type=radio

(or if you want to rebuild the tv cache as well, get_iplayer --cache-rebuild --type=tv,radio)

If there's something wrong with the cache, which is my best guess right now, then this should fix it.
