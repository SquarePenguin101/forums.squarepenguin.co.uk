I don’t know enough about libreelec to know whether you could ever run get-iplayer on it. There are some good instructions on the gip wiki for a generic Linux installation. I suspect you’ll get stuck installing Perl and its dependencies.

The web pvr scripts listen on the localhost port by default - you’ll probably find the setting for this in /etc/default/get_iplayer_web_pvr , but that depends on how gip was packaged for osmc.
