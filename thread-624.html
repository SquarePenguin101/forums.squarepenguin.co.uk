Using latest dev version of get_iplayer and ffmpeg 2.8.4 on Windows 10 I get the following error during conversion from .ts to .mp4 for hlshd (720p25) downloads only
Code:
[h264 @ 04eb1a00] non-existing SPS 0 referenced in buffering period

The error is obviously not that serious as the resulting mp4 files all play perfectly well but I thought I would report it.
