Why are some recordings .ts and some .mp4?

I'm curious. Using V2.99 web interface, some downloads that I've made are .ts files and some are .mp4. Looking at the on-screen log, it seems that some programmes download as "HLS" .ts files and then get converted to .mp4. Looking at the original .ts (making a copy before it gets deleted) and the resulting .mp4, they seem to be identical in encoding and very similar in file size, as if they are essentially the same file in a different wrapper.

MPEG-TS is the native format for the HLS streams.
Quote: Looking at the original .ts (making a copy before it gets deleted) and the resulting .mp4, they seem to be identical in encoding and very similar in file size, as if they are essentially the same file in a different wrapper.
That's for tagging and player compatibility. Use --raw if you only want the .ts file.
