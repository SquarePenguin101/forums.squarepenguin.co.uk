Your settings are configured to record the best available radio quality (320kbps). An hour's recording at this level is approximately 150MB.

The next best setting is 128kbps. An hour at this level normally takes up 60MB.

You need to configure the SETTINGS in your PVR Manager to use a lower than maximum quality.

Refer to these pages for radio modes:

1. https://github.com/get-iplayer/get_iplay...r#settings
2. https://github.com/get-iplayer/get_iplay...odes#radio
3. https://github.com/get-iplayer/get_iplay...adio-modes

You could, of course, set a default lower quality in your Command Line Interface (CLI) using the command:

Quote:get_iplayer --prefs-add --radiomode=better

If you are wanting to end up with an MP3 file, then you're best bet is to download the best audio file at 320kbps M4A and then convert (transcode) to MP3 (with the LAME encoder) using the LAME settings that result in an MP3 of your preferred size.

Transcoding from a 128kbps M4A to MP3 will be worse than going from 320kbps M4A to MP3 (always start with the best source quality). Ideally you'd not want to lossy transcode anyway, but this is your choice.

This is one way to transcode an M4A (AAC) file to an MP3: https://github.com/get-iplayer/get_iplay...i/aactomp3
