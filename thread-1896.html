I'm sure get_iplayer can be confusing if you don't have a technical background

'get_iplayer --purge-files' will delete all programmes over 30 days old if they are still in your download folder - if you've moved them somewhere else, it won't. This is to comply with the BBC requirement that you don't keep programmes for more than 30 days.
To stop this warning ever appearing, use 'get_iplayer --prefs-add --no-purge'
You have to enter these instructions on the command line. If you don't know how to do that and you're using Windows then google "windows command line" or something similar - I can't help you here as I don't run get_iplayer on Windows, nor do I use the PVR.

All this has nothing to do with your cache not refreshing however.
I suggest you read https://forums.squarepenguin.co.uk/thread-655.html and https://forums.squarepenguin.co.uk/thread-706.html
At the very least, please say which version of get_iplayer you're using, how you're running it and any other errors/warning messages you're getting when you're attempting to refresh your cache.
