07-05-2014, 07:01 PM
Windows Phone is a no-go. Phones run on ARM chips and all the various bits of get_iplayer for Windows are x86-based. Phones are locked down, so you wouldn't be able to install without some chicanery anyway. The same considerations apply to Windows tablets. By "Windows OS tablet" I presume you mean an Intel tablet running Windows 8.1. ARM-based tablets and Windows RT wouldn't do the job.
