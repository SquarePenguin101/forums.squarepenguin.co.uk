You can manually tell get_iplayer to refresh your cache using the following command:

Code:
get_iplayer --refresh

It should do it automatically when launched but you may need to do it manually.

This is only a consideration when you're using the Web PVR, as I suspect you are. Doing a one-time refresh from the command line will only help temporarily if that's the case. The first thing you should always do after launching the Web PVR is to click "Refresh Cache". If you are in fact using the CLI rather than the Web PVR, and your cache doesn't update as necessary on launch, then something may be wrong with your settings.
