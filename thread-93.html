get_iplayer's install directory is: C:\Program Files (x86)\get_iplayer

There should be a directory in there called 'MPlayer'. Extract the contents of the 'stand alone' download to that MPlayer directory.

If that MPlayer directory isn't present, create it and place the extracted mplayer-svn-36563 directory inside the MPlayer one.

That should be enough to get it working for you...hopefully.
