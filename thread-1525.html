03-11-2017, 07:56 PM
If you want to restore that spew of "Added:" lines as an option, send a pull request. However, I don't encourage you to waste your valuable time since the functionality is trivial to replace:

Web PVR: Refresh cache, put 1 in "Added Since (hours)" for all programmes added to cache in last hour, i.e., the refresh you just performed, click Search (with default wildcard), see the exact same list of new programmes with added benefit of links to schedule or download. Search for specific programmes as normal to refine results.

CLI: get_iplayer --refresh --since=1 ".*" [other options...] for all programmes added in last hour, i.e., the refresh you just performed, see the exact same list of new programmes, minus the "Added:" prefixes. Use any search terms you like with --since to refine results.
