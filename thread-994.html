12-08-2016, 07:06 PM
That doesn't make sense. Merely moving files around on your computer doesn't delete them from the system, so there is something you left out. It doesn't matter, though. Not to belabour the obvious, but if you don't move the files after download you can delete them in the Web PVR Manager just like any other user. If you want to use a different output directory for your recordings so you don't have to move them yourself, that is covered in the Help page.

Normally, an entry in the recording history without a file would appear dimmed, but it looks like that was broken in the last release. You can still tell if the associated file was deleted or moved because the Actions column will be empty.
