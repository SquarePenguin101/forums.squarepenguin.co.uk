Unfortunately at the moment there are a number of programmes appearing with damaged HLS streams that are missing segments. The examples you have posted are some of these. As hlshd is the only option for 1280x720@25fps streams, you will need to try a different mode.

hlsvhigh might work ok but I haven't checked it. This is an SD option though. If you really want HD then the only option is hvfhd which is 1280x720@50fps. Or you can use hvfsd which is 960x540@50fps and is what iPlayer generally uses as HD.

This has happened before, and it will happen again. Really people - every stream is not going to work for every programme every time, and there is nothing get_iplayer can do about. It can only warn you and stop downloading when it finds a broken stream. It is not a problem with get_iplayer. If you don't want to configure get_iplayer as recommended so you can get fallback streams, that's OK. Just don't expect your broken streams to be mended. The BBC have fixed some in the past, but not many.  Thread closed.

EDIT: This is ultimately a problem with the BBC production pipeline.
