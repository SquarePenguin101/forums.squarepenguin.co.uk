How to show all episodes in a particular radio show

user-1213

19-02-2016, 05:53 PM
So I'm just getting started with get_iplayer.

Win 7 64 home premium.
get_iplayer v2.94

I can download things using --pid or --url, but I'm having problems with searching.

For example I want to see all the episodes in BBC radio 1's residency show.

I use:
Quote:get_iplayer --type=radio --channel="BBC radio 1$" "residency"


which gives me:

Quote:get_iplayer v2.94, Copyright (C) 2008-2010 Phil Lewis
  This program comes with ABSOLUTELY NO WARRANTY; for details use --warranty.
  This is free software, and you are welcome to redistribute it under certain
  conditions; use --conditions for details.

Matches:
10592:  BBC Radio 1's Residency - Steve Angello, BBC Radio 1,
10593:  BBC Radio 1's Residency - Oliver Heldens, BBC Radio 1,


Whereas looking on iplayer, we get: http://www.bbc.co.uk/programmes/b01d76k4...des/player

Which means I'm only seeing the last 2 episodes and not all of them.

Any suggestions?

user-743

19-02-2016, 06:20 PM
Get_iplayer can only show you (and get using the titles/ids) what's in its seven day cache.  For older stuff you have to use the PID or URL.  The programmes you indicate stretch back a while.

user-585

19-02-2016, 07:05 PM
You could also use the brand PID from the search page which is b01d76k4. This will go through all the episodes that have been broadcast and download any that are available (you will need to add --pid-recursive to your command).

There are a lot of episodes in that list though so remember only 10 will actually be successful.
