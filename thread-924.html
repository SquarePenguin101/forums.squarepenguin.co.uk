Greetings - I am using get_iplayer exclusively for radio, so my USA location historically hasn't been a concern in accessing BBC content.

I've used get_iplayer successfully by specifying the PID for programs to capture, but I'm not exactly sure what the correct syntax is for capturing live radio.

The program number for World Service live appears to be 80158, from the liveradio index, but my efforts so far haven't been fruitful:

Code:
80159:  BBC York - live

INFO: 60 Matching Programmes

C:\Program Files\get_iplayer>get_iplayer --type=liveradio 80158

Matches:
80158:  BBC World Service - live

INFO: 1 Matching Programmes

C:\Program Files\get_iplayer>

What am I missing here?  Thanks...Richard in Allentown, PA  USA

It works the same as on-demand radio, i.e., get_iplayer only performs a search by default, as your output shows. You must add --getto record.
