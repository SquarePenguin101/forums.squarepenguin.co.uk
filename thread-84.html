The output you attached doesn't provide good test cases. For Heidi, get_iplayer is telling you that you already downloaded it:

Quote:Already Completed at: 5760.035 sec Duration=5.760 sec

If you delete the previous output file, re-download with
Code:
--force
in your command line. The above message indicates the file still exists.

For Newsround, you are trying to download an episode that is no longer available from iPlayer, as the get_iplayer output indicates:

Quote:WARNING: No programmes are available for this pid with version(s): default

Daily news programmes generally disappear after 24 hours.

If you do more testing, find a programme that you've never downloaded, verify it is available on the iPlayer site, then download it with the
Code:
--verbose
option in your command line to capture all available output.

16-12-2013, 05:30 PM
To answer your other question: yes, there is a system preferences file separate from user prefs. However, you should never touch that file - it is managed by the installer. Your output doesn't indicate any problems with preferences.
