Can't download even using pid

user-890

26-03-2015, 11:58 AM
I've been using get-iplayer very successfully for some years and I'm using 2.92 on Ubuntu 14.10 at the moment.

I'm often failing to get expected downloads; I can do a
Code:
get-iplayer --pvr
and it will collect the tv listings and populate the tv.cache file but none of the expected programmes gets downloaded (even though they appear to be present in tv.cache)

If I go to the BBC website and get a PID and try to do get-iplayer
Code:
--pid b012345
then nothing gets downloaded - no message saying why, just no download.

If I delete tv.cache then the programme will download as expected.

I'm guessing that something is messed up in tv.cache but even if I delete it and run
Code:
get-iplayer --pid
to get a different programme it won't download.

I had thought the problem was the download_history file had got too big (c. 4 years worth of downloads) but even if I delete it I still get some odd problems.

Any ideas?
