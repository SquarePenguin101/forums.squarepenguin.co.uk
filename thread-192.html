Since you are trying to access iPlayer from outside the UK, we can't help you, so I'm going to close this thread. However, here are a few general suggestions:

1. Your browser screen shot suggests that you do not have the latest version, or else your cache hasn't been updated for nearly 5 months. If it doesn't say version 2.86 at the bottom of the web pvr screen, then you don't have the latest version

2. Stop using the web pvr to debug your problem. It only obfuscates what may be going on and is useless for debugging. Open the get_iplayer console window and attempt to refresh the cache with the following command:

Code:
get_iplayer --type tv,radio --refresh --verbose

Assuming it runs, look for error messages in the rather voluminous output.

3. Your problem with invoking perl.exe indicates something is wrong with your system - a bad/missing/misinstalled DLL or something like that. Nobody else will know what you've done to your system, so it will be impossible to even guess. All I can suggest is that if you've installed a non-get_iplayer version of Perl at any time, remove it completely and do a complete uninstall/re-install of get_iplayer. And make sure you install get_iplayer while logged in as a user who is a member of the Adminstrators group.
