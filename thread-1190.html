For everyone else: Don't delete your cache files. It would be like relieving an itch on your elbow by cutting off your arm. You'll have to update for another month to rebuild a 30-day cache. Cache files are plain text, so if you suspect that something has corrupted one (very rare, but not impossible), open it in a proper text editor like Notepad++ and excise the offending bits, then force a refresh. And in general, don't base any decisions on what happens in the Web PVR - it can't be trusted. Break out to the command prompt to test searches, cache refreshes, etc.
