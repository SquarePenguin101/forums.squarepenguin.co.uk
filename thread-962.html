That is by design. Brand/series images are now used instead of episode frame grabs. I neglected to add this change to the 2.95 release notes - now corrected at GitHub.

user-1346

21-07-2016, 01:38 PM
Okay. Is there any way to force the program to download the episode thumbnail rather than the series one?

user-2

21-07-2016, 02:21 PM
No. However, if you want the image for a particular show you can find it in the HTML source of its iPlayer episode page (e.g., http://www.bbc.co.uk/iplayer/episode/p04...-6-memoirs). Search for "og:image", which will locate this:

<meta property="og:image" content="http://ichef.bbci.co.uk/images/ic/1200x675/p042112x.jpg"/>
