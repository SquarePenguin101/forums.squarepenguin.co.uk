There is an 'issue' with email notifications. MyBB, by default, has a policy whereby you receive a notification and then until you visit again, you'll receive no further notifications.

I can understand this policy, particularly if board admins have defaulted everyone to receive email notifications on every new thread or something as it is probably in place to stop appearing spammy.

But, in these forums, at registration I leave everything regarding notification to off and you have to opt in. If you post a thread then the default is, as the thread poster, you will receive an email notification that a reply has been made - for that thread you started ONLY.

To receive notifications for all new threads or posts as a 'normal' member you have to opt in. I don't want to spam you but to my mind, if you opt in for a notification then you expect a notification.

To that end, I'll be changing the default behaviour so that if you subscribe then you'll get notifications immediately upon new threads and/or posts depending on what you selected.

As I said, the default is no notifications so you'd have to opt in to this - I'm cautious of sending annoying emails that aren't wanted so it'll always be opt in.

Lastly, there is currently the option at registration to receive notifications via PM. That'll be removed as PM functionality is disabled here.

I'll make these changes as soon as time constraints and other commitments allow but they should be done before the weekend is over.
