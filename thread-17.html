The web site might have been down temporarily. Run the installer again and select the get_iplayer component for re-installation. Then run get_iplayer again. If you still get the same error, create this directory:
Code:
C:\Users\<your-username>\.get_iplayer\plugins


where
Code:
<your-username>
is just what it says. Note the dot before "get_iplayer". Creating the directory will only stop the error message. It won't actually fix the problem, so run get_iplayer again to see if it can download the plugins with that directory already created. This should happen during installation, but either the web site was down when you ran the installer or you have some kind of permissions problem in your home directory.
