user-493

09-08-2016, 10:44 PM
Yes this is a current issue affecting HLS downloads , also documented in the Linux section of this forum (the segmented nature of this forum doesn't help these sorts of problems), it seems to be a problem at the BBC end of things.

For now using flash, eg --tvmode=flashhd will produce fault free downloads , although we don't know how long this will be available. For me, HLSHD downloads are between 30 and 100 Mbps - be warned that FlashHd is much slower, four times slower for me! :-(

The new hvf mode is another way around the problem although I would recommend reading around the subject.

I'm download using a batch file which I've amended to pipe results into a text error file - I then examine the output and where there are missing segments I repeat the download using flash.

Hope this helps

Alan
