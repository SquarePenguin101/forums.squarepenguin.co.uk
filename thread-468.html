Using get_iplayer (GiP) through Get iPlayer Automator (GiA), it is possible to download ITV programs.  Yet I have not found a way to get GiP itself to find or get ITV programs.  I've found conflicting information on the web about whether GiP supports ITV, with everything from a categorical "no" to specific instructions on how to do it (--type=itv).  I know ITV made some changes that broke GiP's support for ITV at one point, but I also know I routinely download Heartbeat using GiA, as recently as today.
