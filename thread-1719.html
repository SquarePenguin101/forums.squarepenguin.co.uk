How do I use --available-since?

user-721

25-03-2018, 12:51 PM
Hi folks,

Re: get_iplayer 3.13.0 on Win10 64

There aren't any new posts about this which probably means I'm missing something obvious, but to return programmes added in the last 24 hours I can no longer get this command to work:

get_iplayer --available-since 24

The result I get is:

ERROR: Search term(s) required. To list all programmes, use ".*" (incl. quotes)

get_iplayer ".*" correctly returns all programmes but can someone please give me the correct command to return a given number of hours? I've tried adding inverted commas and full stop in every combination I can think of, all without success.

Many thanks and apologies for wasting your time if I am having a senior moment

user-721

25-03-2018, 01:50 PM
(25-03-2018, 01:28 PM)get_iplayer.cmd ".*"  --future  --since 24

Thank you VERY much kind sir, much appreciated!

user-2

25-03-2018, 05:03 PM
Bear in mind that is the form but not the function. get_iplayer ".*"  --future  --since 24 is not the same as get_iplayer ".*"  --available-since 24
