Yes, it's a local problem, but only you will know what you have done to your system. Â Check if these files exist:

C:\Program Files (x86)\get_iplayer\lib\auto\XML\LibXML\LibXML.dll

If they do, check that their permissions allow them to be accessed by you. If so, then I can't offer any further suggestions. You're not the first to encounter this problem, but it is not reproducible. Nobody has found a solution, other than perhaps running get_iplayer as administrator, which is not a very good solution.
