Hi mumbo,
get_iplayer used to be able to index red button content, but that ability was lost when the BBC removed the index feeds that GiP used to create the cache.

The feeds GiP now uses are the BBC's schedule feeds which contain a lot less info than the old feeds did.

Some of the things that were lost were category info, web-only content and red button listings.

I don't think there is a source for these now.
