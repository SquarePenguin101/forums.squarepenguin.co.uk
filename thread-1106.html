(29-11-2016, 06:33 PM)You were absolutely correct. I was still on version 2.95 which is strange as I used the get_iplayer_setup_latest.exe to download and install what I thought was the latest version. I assume this executable only updates the get_iplayer not the Web Manager?
It updates both. Installers have the version number in the file name since 2.95. You used an obsolete installer from somewhere, so it wouldn't have updated your machine.
Quote:The effected entries are greyed out and most are where the original files have been moved or deleted Is there an ini or xml file that holds these "Phantom Files" that can be edited to remove their entries?
Greyed out means the associated file has been moved or deleted. You can't delete history entries from the Web PVR Manager. You must edit the file C:\Users\<username>\.get_iplayer\download_history. However, there is no reason to delete those entries.
