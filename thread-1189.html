I would never implement such a feature since it essentially duplicates PVR queuing. I would also be extremely reluctant to accept a pull request for the same reason, and also because it would be a support headache. I can't be in the position of debugging iPlayer web applications. If you or anyone else wants to have a crack at it, I suggest you create a separate utility that sucks down "Listen Later" listings and queues them for the get_iplayer PVR.
