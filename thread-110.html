Hi,
I downloaded a program, but it turned out to be only the half of it.
SoI tried to download it again, but the terminal says I need to 'use --force to override'.
How can I do that?

user-30

08-02-2014, 11:11 AM
You can simply run the command again but append
Code:
--force
to the end of it, e.g.

Code:
get_iplayer --get 123 --force

This overrides the download history and will allow you to re-download a previously downloaded programme.
