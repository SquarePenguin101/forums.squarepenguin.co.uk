The guidelines for a Windows install are here:

The direct download link to the recently released v2.96 executable file is
https://github.com/get-iplayer/get_iplay...ses/latest

Download the get_iplayer-2.96.0.exe to a location where you'll find it again (usually in Windows ends up in the 'Downloads' folder) and run the file to install get_iplayer.
