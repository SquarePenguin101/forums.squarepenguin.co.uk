Ignore the Versions column in search results. It is a useless vestigial appendage, and has been for a long time. It is no longer displayed by default, and you should deselect it in the Columns tab so it isn't visible. The only way to know what programme versions are available is to click on the episode or description to see the detailed info page. Never specify an explicit value in the "Programme Versions" field of the Recording tab unless you know what you are doing, or you specifically want audiodescribed or signed programmes. "default" is not a literal version specifier. It is basically a placeholder for whatever non-audiodescribed, non-signed version(s) may be available. It should never be explicitly specified on its own - it is unnecessary.
