Hi,
Sorry if this is a trivial question (Win 10 PC).

I record mostly radio4 radio shows to copy to mp3 CD for the missus to listen to in her car.
As radio4 radio repeats shows often I find it useful to have get_iplayer report that I've already got it (i.e. 'Already in history').

I'd like to be able to backup and restore the history file as required (new PC, new HDD etc.).
Is the History file this?...

c:\Users\Andrew\.get_iplayer\download_history

...and if so, can I simply take a copy of the above file to get a backup (and restore if needed)?

Thanks,

Andrew.

Yes, if you open it with a text editor, you will see all the relevant information relating to the download, such as PID, programme title, when downloaded, the mode used, the programme details, etc.

If you use the PVR then you might also want to copy over the pvr folder in .get_iplayer as that contains the list of programmes you set for download by the PVR.
