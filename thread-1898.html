"You have programmes over 30 days old that should be deleted" but there aren't any

user-2264

05-10-2018, 09:55 PM
Hello everyone,

I get the following:
WARNING: You have programmes over 30 days old that should be deleted.
WARNING: Find them with 'get_iplayer --history --before=720 ".*"'
WARNING: or use the 'Recordings' tab in the Web PVR Manager.
WARNING: Use 'get_iplayer --purge-files' to delete all programmes over 30 days old.
WARNING: Use 'get_iplayer --prefs-add --no-purge' to suppress this warning.

But the folder to which get_iplayer downloads files is empty, so this seems unlikely. When I run 'get_iplayer --history --before=720 ".*"' it displays a list of programme names, but no filepaths. How can I get the filepaths of any programmes which are causing the message to display?

user-2

05-10-2018, 11:02 PM
You don't need to know the file paths to purge them. If you feel a need to see them, add --listformat="<filename>" --skipdeleted to the history search.

user-2264

05-10-2018, 11:37 PM
Thank you! That did the trick, I found and removed the offending file.
