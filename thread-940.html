If you don't understand it then it's probably best to not use it. That's not supposed to be an elitist sentiment, but you need to have the ability to safely follow the instructions if you are to use the development version.

There is no easy one click installer like the general release. If you want to install the development version you'll need to follow the instructions as @tvfan linked.

Those are the instructions for installing it, no one's holding anything back from you, you just need to understand and follow them if you want to use it.

If you want to press on, you'll need to get up to speed with git at the least and you can start that at the link below:

https://git-scm.com/book/en/v2

I suppose you could download and replace the core script without git but again - you'd need to fully understand what you are doing and there aren't any instructions written up for that.
