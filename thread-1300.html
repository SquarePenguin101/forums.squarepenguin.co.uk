For everyone else: If you are not accustomed to managing your own Perl module library, I recommend you first try the Mojolicious package provided by your distro, if one exists. It may be old but still work OK with get_iplayer. Your distro's Mojolicious package may pull in some optional dependencies that can speed up programme indexing. If your distro's package doesn't work, you can always uninstall it and install Mojolicious from CPAN as suggested here.
