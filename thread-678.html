Actually, if you want to do that renaming, do it as part of the recording command or save it as a preference. If you read on from that section you read in the guide, you'd see it was used as part of a larger command.

You could add the renaming as a preference by using a command such as:

get_iplayer --prefs-add --file-prefix="<nameshort><-senum><-episodeshort>"
