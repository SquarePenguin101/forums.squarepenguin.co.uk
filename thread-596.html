get_iplayer Output:

On windows (10) 64 bit it will generally install itself
c:\Program Files (x86)\get_iplayer
or
32 bit
c:\Program Files\get_iplayer

The main script is
c:\Program Files (x86)\get_iplayer>get_iplayer.cmd

If you knew that today's shipping forecast was on
http://www.bbc.co.uk/programmes/b06r0dgr

you'd enter the command
c:\Program Files (x86)\get_iplayer>get_iplayer.cmd --type=radio --pid=b06r0dgr

Best read the wiki for how to change the output format, use the PVR and many other features.
https://squarepenguin.co.uk/wiki/
