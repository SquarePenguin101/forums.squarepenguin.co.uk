I can answer my own question:

Run this command:

get_iplayer --prefs-add --aactomp3

in the get_iplayer install directory and it sets it for you.

I still don't know how to do it in the web pvr window though.

Hi Alan,

The radio download guide has a section on this:

https://squarepenguin.co.uk/guides/radio...mes-to-mp3

I don't think there is a switch in the webpvr itself to set this preference, you'd set it in the command live version.

See in the guide that it gives an example command to do this for a one off download (but you'd need to use the command line to perform the download) and an example of how to add this as a preference to automatically convert all downloads to mp3.

When you add preferences they are remembered in the command line AND webpvr as the webpvr is really just calling the get_iplayer command line version in the background to perform its tasks.

So for your situation I'd recommend adding the preference and the webpvr will set the download to be converted.
