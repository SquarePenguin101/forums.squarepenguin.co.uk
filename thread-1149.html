(02-02-2017, 09:23 PM)I have tried cleaning up the "old" version repeatedly (because it says on installation that it was installed for a previous user) but can't see what else on my system is still befing referenced - I have deleted all files, edited registry entries and still it refers to a previous version.
That's not what it says. It only warns you that if you're installing under a different account than you normally use with get_iplayer that you will need to complete the cleanup yourself. The whole point is that the installer doesn't know if the previous installation was done under a different account.

I have no idea what you mean by "shows files being identified", so you had better post a screenshot to show what is happening. You should also check whether or not you've merely screwed up your settings. Post the output from get_iplayer --show-options, and also try to do a search from the command prompt, e.g., get_iplayer Taboo. If that doesn't return results, the problem is in get_iplayer, not the Web PVR.
