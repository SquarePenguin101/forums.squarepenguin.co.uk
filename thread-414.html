(28-07-2015, 11:21 AM)This is what I am entering in the terminal

get-iplayer --type=radio --pid=b01m0pqk

and this does not produce ANY "partial file" in my usual download location.

It does produce that file, because that's how get_iplayer works. It downloads the file as a .partial.m4a.flv file and it'll then convert that to a standard m4a file. Sometimes, when downloads get interrupted the .partial.m4a.flv file can get left behind and become 'corrupted' which means the download can't be resumed.

If you try a download when that partial file is there then get_iplayer will try and restart the download using that file. To start over and re-download the whole thing you need to go and delete that partial file to allow get_iplayer to do a clean download.
