Is there a valid list of fields for use with --listformat?

user-1812

29-06-2017, 02:50 PM
Is there a list of valid fields that can be used with the --listformat command anywhere? I've checked the documentation but can't see anything specific.

I can see all the fields that appear to exist when running a --info command, but when I try use the three I want (senum, desclong & firstbcastdate) the returning string for those fields is just the fieldname itself.

Any help greatly appreciated.

user-2

29-06-2017, 02:55 PM
--listformat uses fields from programme index cache, not programme metadata. See first line of tv.cache file for field names. See FAQ list for tv.cache location.
