Sure, I can download with get_iplayer using Linux on my PC; then view the file remotely (eg using Kodi) on the android device, or transfer the file to the android device (various ways). But I was curious to know how to use get_iplayer directly on an android device. I could not find anything on the wibbly wobbly web, so the lateral thinking began .... (this is a bit slow in my 68-year old brain).

Here we go:
My android device has a ARM processor.
Raspberry Pi (doesn't do android) has an ARM processor.
Raspbian is Debian (Wheezy).
Android app "Debian No Root" is debian (wheezy).
Nice instructions here [url=http://squarepenguin.co.uk/wiki/installation/raspbian/][/url]
on how to install get_iplayer on raspbian/raspberry pi.
(This is logical, Captain.)

Anyway, it works! See attached screenshots. I have get_iplayer 2.91, the CLI and the Web PVR working fine.

It is not difficult, but there are prerequisites and precautions which I can explain.
I have no idea whether anyone else is interested, so if this post gets some responses I will sharpen my quill pen and provide details.
