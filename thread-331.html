Searching working on Windows but not Linux?

user-829

16-01-2015, 05:09 PM
Hi, I installed get_iplayer 2.91 for Windows, which is working flawlessly. However, having installed 2.91 for Ubuntu from the PPA, all it will return is the "BBC iPlayer Feeds" RSS failure...

I feel like this is a very obvious mistake, a missing package or something, but I can't seem to track it down. Any ideas?

user-2

16-01-2015, 06:20 PM
Force a cache refresh:

Code:
get_player --refresh --type tv,radio



If that doesn't work, make sure you're actually running 2.91:

Code:
get_player -V

user-2

16-01-2015, 06:21 PM
That should be:

Code:
get_player --refresh --type tv,radio,livetv,liveradio
