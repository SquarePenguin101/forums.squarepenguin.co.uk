get_iplayer search and PVR functions no longer work - no fix available

user-2

29-10-2014, 02:16 PM
The BBC have removed the programme data feeds used by get_iplayer, so search and PVR functions for TV and radio no longer work.  There is no programme information to cache, and it was the cache that supported search and PVR functions.  There is no fix available at this time.  You can still download individual programmes via PID or URL.  Podcast searches still work.

http://iplayerhelp.external.bbc.co.uk/tv/feeds
