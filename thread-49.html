First time using get_iPlayer, I'm running Vista. I got this error message when specifying flashhigh mode:

Warning: Required rtmpdump does not exist - cannot download Flash audio/video.
Info: skipping flashhigh1 mode
Error: Failed to record

So I tried specifying every single mode in turn, and got errors each time. The flashxxx modes gave the above error message. The rtspxxx modes gave the following error message:

Info: Recording Low Quality H.264 stream
'cvlc' is not recognized as an internal or external command, operable program or batch file.

Info: skipping rtspstd1 mode
Error: Failed to record

Finally specifying subtitles mode records the .srt file containing the text of the subtitles, but doesn't download anything else, so I still don't have a playable video file.
So what else does get_iplayer need to be able to record? And are these problems at all related to the BBC's recent revamp (27th Sep) of their iplayer website?
