Hi. Long time user (It's Great!), first time poster.

My "workflow" was to look up the latest on BBC Sounds and then search for a keyword using get_iplayer. (I doubt I am alone in this, but I included for clarity)

There have been some changes to the way that BBC sounds displays, say, comedy such that it includes the latest in sub-categories etc. ie not chronological any more.

This doesn't suit me at all. (Why did they mess with the original rather than just adding additional features?!)

Is there any way to instantiate categories in Get_iplayer such that I can use this feature to get a chrono summary of the recent comedy radio releases? Or do you have another solution?

Thanks and Very Best Regards

Richard.

I looked at the format for other links in BBC Sounds and found I could do this

.www.bbc.co.uk/sounds/category/comedy-sketch?sort=latest

ie I added "-sketch" to the new comedy link and have achieved something of the result I wanted.
