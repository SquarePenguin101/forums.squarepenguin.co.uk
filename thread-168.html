I upgraded to Get_iplayer v2.86 a couple of weeks ago.

I've got s series of radio programmes and podcasts which I download nightly with a scheduled run of get_iplayer--pvr.

I've noticed odd problems but I've now found that all the podcasts downloaded since the upgrade are unplayable.

I'm running Windows7 64 Bit.

I've done some troubleshooting on this and I've found that then the .mp3 file is being downloaded from the web site that the progress indicator on the screen shows that it is downloading ~ 150% of the original mp3 file.

I can manually download the files using my web browser from the BBC websites using the URLs given and the files saved this way have the correct size and play in vlc.
